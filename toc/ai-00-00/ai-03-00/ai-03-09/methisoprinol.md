# Methisoprinol

## Imin 500mg

##### 

| 標題       | 說明                                                          |
|:-----------|:--------------------------------------------------------------|
| 藥品代碼   | OIMIN                                                         |
| 適應症     | Viral disease, immunomodulator.                               |
| 用法用量   | Adult: 500 mg q3-4h. Children: 50 mg/kg/day in divided doses. |
| 肝功能異常 | 無需調整劑量                                                  |
| 腎功能異常 | 無需調整劑量                                                  |
| 禁忌       | No Data                                                       |
| 副作用     | GI discomfort. Transient rise in serum & uric acid levels.    |
| 孕期建議   | No Data                                                       |
| 哺乳期建議 | Contraindicated 哺乳期使用禁忌                                |
